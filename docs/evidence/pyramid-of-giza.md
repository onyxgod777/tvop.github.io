# Pyramid of Giza

![Great Pyramid](/img/great-pyramid.jpg){ align=right width=250 }

The Great Pyramid of Giza — the last surviving wonder of the ancient world — contains precise geometric encoding of both the Golden Ratio and Golden Pi within its dimensions.

## The Numbers

| Measurement | Value | Ratio |
|-------------|-------|-------|
| Base side length | 440 cubits | — |
| Original height | 280 cubits | — |
| Slant height | 356 cubits | — |
| Base / Height | 440/280 = 11/7 | ≈ 1.5714 |
| Slant / Half-base | 356/220 | ≈ Φ (1.618) |
| Height / Half-base | 280/220 = 14/11 | ≈ 1.2727 ≈ √Φ |

## Golden Pi Connection

The pyramid's slope angle corresponds to a rise of 14 units for every 11 units of run. This ratio of **14/11** is an excellent approximation of **√Φ ≈ 1.27202** — the key constant in the Golden Pi formula.

When a circle is drawn with radius equal to the pyramid's height, its circumference relates to the pyramid's base perimeter through the Golden Pi constant.

## Royal Cubit

The pyramid was built using the Royal Cubit, which itself encodes the relationship between Φ and π:

$$\text{Royal Cubit} = \frac{\Phi^2}{5} = \frac{\pi}{6} \approx 0.5236 \text{ m}$$
