The conventional value of π = 3.14159... is the *approximate* ratio used in engineering. **Golden Pi** proposes that the *true* geometric value is:


$$
\pi = \frac{4}{\sqrt{\Phi}} = 3.144605511029693\ldots
$$
## The Formula
| Constant | Value |
| --- | --- |
| Φ (Phi) | 1.618033988749895... |
| √Φ | 1.272019649514069... |
| π = 4/√Φ | 3.144605511029693... |
## Derivation
From [Kepler's Right Triangle](kepler-triangle.md), where the sides are in the ratio $1 : \sqrt{\Phi} : \Phi$, the circumference of a circle with diameter equal to the triangle's height relates to $4/\sqrt{\Phi}$.


## How It Differs from Conventional Pi
| Aspect | Conventional π (3.14159) | Golden Pi (3.14460) |
| --- | --- | --- |
| Source | Archimedes approximation | Geometric derivation from Φ |
| Precision | Engineering limit | Geometric exactness |
| Squaring the Circle | Impossible (transcendental) | Solvable geometrically |
| Connection to Φ | None | Direct via √Φ |
