# Instrumentum

The **Instrumentum** is a collection of mathematical tools, formulas, and references for working with Golden Pi (π = 4/√φ).

## Core Identity

$$\pi = \frac{4}{\sqrt{\Phi}} \approx 3.144605511029693$$

## Key Relationships

### Squaring the Circle

The perimeter of a square equals the circumference of a circle when:

$$4s = 2\pi r$$

With Golden Pi: when \(r = \sqrt{\Phi}\) and \(s = 2\), the equation holds exactly.

### φ² + 1/φ² = 3
### φ⁴ + 1/φ⁴ = 7

These identities reveal that the numbers 3 and 7 are hidden within the Golden Ratio.

### Kepler's Right Triangle

A right triangle with sides in geometric progression: $1 : \sqrt{\Phi} : \Phi$

This triangle provides the direct geometric link between φ and the true value of π.

## Minimal Polynomial

Given \(a = 3\) and \(b = 7\):

- \(m = a + b = 10\)
- \(n = a - b = -4\)
- \(\alpha = m + n = 6\)
- \(\beta = m \times n = -40\)

Then:

$$f(x) = x^4 + \alpha x^2 + \beta = 0$$

## Field Relationships

Golden Pi \(4/\sqrt{\Phi}\) lives in the quadratic field \(\mathbb{Q}(\sqrt{5})\), the same field as the Golden Ratio — making it algebraic, not transcendental.
